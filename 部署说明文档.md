以下配置操作前，确保要有idea，jdk17, maven3.6.1, mysql ,navicat ,node14 ，相关环境！！！

第一步：把源码文件的springboot文件夹拖进idea里面
第二步：设置里面配置maven,把maven导入
第三步：选择jdk17的Java环境
第四步：navicat连接mysql,把数据库文件导入（农产品销售系统\源码\springboot\src\main\resources\application.yml里面修改你的数据库库密码）
第五步：springbootApplication类中启动后端
第六步：在idea里面通过dev启动前端，
              或者在农产品销售系统\源码\vue这个目录下面输入cmd,命令行里面输入npm run serve启动前端，
              或者把vue文件夹拖到VScode里面，在终端里面输入npm run serve启动前端，
